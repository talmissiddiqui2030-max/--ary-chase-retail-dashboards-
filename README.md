# ARY Chase Retail Analytics Dashboards

This repository showcases interactive dashboards built in **Power BI** and **Excel (Power Query + Pivot Tables)** for ARY Chase retail analytics.  
The dashboards cover **Revenue, Profit, Returns, and Daily Sales Trends**, designed to support business decision-making.
